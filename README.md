<summary><b>🇬🇧 English Description</b></summary>

A simple Minecraft mod designed for quickly pasting pre-defined text snippets that are otherwise tedious to type out manually in chat. Perfect for Vanilla, SMP servers, or lightweight Vanilla+ modpacks.

Install and go—zero complex configurations required.

*(Supports both auto-paste functionality and a "save once, use forever" system).*

*The name is a blend of Yahoo + Paste.*

---

> **Note:** This mod is only tested locally and strictly on **Fabric 1.21.11**. Compatibility with other versions is rarely tested (or uploaded purely on luck).
> 
> Developed purely as a hobby in my free time, so frequent updates or long-term support are not guaranteed.
>
> This project is 99.9% written with the help of AI. I don't officially know Java, so if any complex bugs arise, they might take a long time to fix (or might not be fixed at all).

---

### 🛠 How to open the menu?

The menu automatically appears in the **bottom-right corner** of your screen whenever you open the in-game chat (**[T]**).
