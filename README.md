# Yahooste

<img align="right" width="300" src="https://github.com/rcatlovelua/Yahooste/blob/main/.not-for-mod-srcc/yahoostee.jpg?raw=true">

Простой мод для Minecraft, чтобы быстро вставлять текст, который обычно долго вводить в чат. Полезно для ванилы, SMP-серверов или легких сборок (Vanilla+).

Поставил и пользуешься — никаких сложных настроек тут просто нет.

*(Есть как авто-вставка, так и тема «один раз ввёл — используешь всегда»)*

*Название собрано из Yahoo + Paste.*

---

> [!WARNING]
> Мод тестируется только на моем компе и строго на версии **Fabric 1.21.11**. Все остальные версии практически не тестируются (или вообще выкладываются на удачу).

> [!NOTE]
> Делаю это чисто от скуки в свободное время, поэтому частая поддержка не гарантируется.

> [!NOTE]
> Мод на 99.9% написан через нейронки. Сам я Java не знаю, так что если вылезет сложный баг, чинить я его буду долго (или никогда).

---

## 🛠 Как открыть меню?

> [!IMPORTANT]
> Меню открывается по открытию чата (**[T]**)  и показывается в нижнем правом углу экрана.
